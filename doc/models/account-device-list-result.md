
# Account Device List Result

Response for a request to list down account devices.

## Structure

`AccountDeviceListResult`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `devices` | [`Array<ThingspaceDevice>`](../../doc/models/thingspace-device.md) | Optional | Up to 10,000 devices that you want to move to a different account, specified by device identifier. |
| `has_more_data` | `TrueClass \| FalseClass` | Optional | False for a status 200 response.True for a status 202 response, indicating that there is more data to be retrieved. |
