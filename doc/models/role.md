
# Role

Role of the user calling API.

## Structure

`Role`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `account_id` | `String` | Optional | Account ID of the role. |
| `external_id` | `String` | Optional | External ID of the role. |
| `role_arn` | `String` | Optional | Role ARN of the role. |
