
# Custom Fields

Custom data that can be included using key-value pairs.

## Structure

`CustomFields`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `key` | `String` | Required | The key for an extended attribute. |
| `value` | `String` | Required | The value of an extended attribute. |

## Example (as JSON)

```json
{
  "key": "CustomField2",
  "value": "SuperVend"
}
```
