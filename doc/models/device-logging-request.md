
# Device Logging Request

Device logging information.

## Structure

`DeviceLoggingRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `device_ids` | `Array<String>` | Required | List of device IMEI identifiers. |
